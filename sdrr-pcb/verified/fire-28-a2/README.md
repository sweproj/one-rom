# One ROM Fire 28 pin 

A2 revision of the 28 pin Fire ROM board.  Combined USB/Pro version with USB-C and SWD header pins.

Minor differences from revision A:
- Programming pins have been slightly rearranged.

## Errata

The silkscreen on this revision mislabels the SWD CLK and DIO pins.  The CLK is DIO and DIO is CLK.
